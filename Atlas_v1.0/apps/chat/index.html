Setup Guide

This app runs as a downloaded HTML file. Follow the steps below for your browser to enable notifications and microphone access.

Google Chrome
  1. Open chrome://settings/content/notifications in the address bar
  2. Under Allowed to send notifications, click Add
  3. Type file:/// and click Add
  4. For microphone: open chrome://settings/content/microphone → add file:///
Microsoft Edge
  1. Open edge://settings/content/notifications
  2. Under Allow, click Add → type file:/// → click Add
  3. For microphone: edge://settings/content/microphone → add file:///
Firefox
  1. When prompted, click Allow in the notification permission bar
  2. If you missed it: Menu → Settings → Privacy & Security
  3. Scroll to Permissions → Notifications → Settings
  4. Find file:///, set status to Allow
Opera / Opera GX
  1. Open opera://settings/content/notifications
  2. Under Allowed, click Add → type file:/// → click Add
Safari — Not Supported

Safari blocks notifications and microphone access for local files. Use Chrome, Edge, Firefox, or Opera.

Tip: Save this file to your Desktop or Documents folder — not Google Drive or iCloud — so messages stay saved between sessions.
